# Sistema de Monitoramento Agrícola por Imagens de Satélite

Projeto de TCC do curso de Ciência da Computação.

Objetivo:
Desenvolver um sistema capaz de analisar imagens de satélite para monitoramento agrícola,
utilizando índices de vegetação, dados climáticos e detecção de anomalias.

Tecnologias previstas:

- Python
- Streamlit
- Google Earth Engine
- APIs climáticas
- Processamento geoespacial

Status:
Em desenvolvimento.
